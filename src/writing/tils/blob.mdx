TLDR: a `Blob` is raw data with a `size` (in bytes) and a (MIME) `type`.

```ts blob.ts
const str = "pete";
const blob = new Blob(["pete"]);
console.log(blob);

/*
  {
    _buffer: { '0': 112, '1': 101, '2': 116, '3': 101 },
    size: 4,
    type: ''
  }
*/
```

That `_buffer` contains the bytes for each character: 112 is the ASCII codepoint for 'p', 101 for 'e', and 116 for 't'.

I added [`Blob`](https://developer.mozilla.org/en-US/docs/Web/API/Blob) [polyfill](https://github.com/eligrey/Blob.js/blob/master/Blob.js) to the Membrane JS runtime (built on top of QuickJS).
