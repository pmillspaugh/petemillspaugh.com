To control scroll behavior when reaching the boundary of a scrolling area, use [`overscroll-behavior`](https://developer.mozilla.org/en-US/docs/Web/CSS/overscroll-behavior).

```css styles.css
body {
  overscroll-behavior: auto; /* default */
  overscroll-behavior: contain;
  overscroll-behavior: none;
}
```

I used `overscroll-behavior: none;` to prevent pull-to-refresh behavior on mobile browsers for my <Link href="/edison-bulb">edison bulb night mode</Link>.

As of 2023, it is [well-supported in browsers](https://caniuse.com/?search=overscroll-behavior).
